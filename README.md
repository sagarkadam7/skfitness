# SK Fitness — Online Coaching Website

Premium online fitness coaching website for **Sagar Kadam**.

## Features

Based on research from top platforms (Trainerize, ProCoach, NOWFit, CoreCoach):

- **Hero & social proof** — stats, trust badges, strong CTAs
- **About coach** — credentials and personal story
- **Programs** — fat loss, muscle building, 1-on-1 coaching, nutrition
- **Pricing tiers** — Starter, Premium, Elite with feature comparison
- **Client transformations** — before/after results with testimonials
- **How it works** — 4-step onboarding flow
- **Free tools** — BMI & macro calculators (lead magnets)
- **Booking form** — discovery call intake with goals, experience, injuries
- **FAQ** — common coaching questions
- **Newsletter** — email capture
- **WhatsApp float** — instant contact button
- **Mobile-first** — responsive dark athletic design

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Customize

Edit `src/lib/data.ts` to update:

- Contact info (email, phone, WhatsApp)
- Social media links
- Pricing and program details
- Testimonials and transformations

Replace placeholder coach photo in `About.tsx` with your own image.

## Deploy

Deploy to [Vercel](https://vercel.com) or any Next.js host:

```bash
npm run build
npm start
```
